# 🛍️ Product Classification & Anomaly Detection

## 📌 Overview
This project focuses on **classifying products** and **detecting anomalies (defective or low-quality items)** using Machine Learning techniques in Python. It helps improve product quality by identifying unusual patterns in data.

---

## 🚀 Features
- 📦 Product classification using ML algorithms
- ⚠️ Anomaly detection for quality control
- 📊 Data preprocessing and feature engineering
- 📈 Model training and evaluation
- 🧠 Supports multiple ML models

---

## 🛠️ Technologies Used
- Python 🐍
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
